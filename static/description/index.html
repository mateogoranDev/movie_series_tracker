📦 Instalación

  1. Copia el módulo en tu carpeta de addons.
  2. Actualiza la lista de módulos en Apps.
  3. Busca "Cine Tracker" e instala.
  4. Configura tu API Key de TMDB.

🔑 TMDB API

  • Obtén tu clave gratuita en themoviedb.org
  • Configúrala en Ajustes → Cine Tracker
  • El banner de configuración desaparece automáticamente
